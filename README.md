# Advanced-Firewall

Python-based firewall system for monitoring and filtering network traffic.

## Features
- IP and port filtering
- Protocol control (TCP, UDP)
- Traffic logging
- Alerts for suspicious activity
- Configurable rules

## Installation
```bash
git clone https://github.com/<username>/Advanced-Firewall.git
cd Advanced-Firewall
